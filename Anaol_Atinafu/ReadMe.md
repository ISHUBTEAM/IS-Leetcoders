## 🚀 How to Contribute

1. 🍴 Fork this repository
2. ✅ Solve a LeetCode problem
3. 📂 Save your solution in `solutions/`  
   - Filename format: `yourusername_problem-name.py`
4. 📝 Create a pull request once in a week to share with the community
